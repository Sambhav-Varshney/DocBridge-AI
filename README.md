# DocBridge AI
A modern SaaS full-stack doctor appointment booking platform with an AI symptom checker.

## Features
- **Frontend**: Clean, response UI built with HTML/CSS and glassmorphism design.
- **Backend**: RESTful API powered by Node.js and Express.
- **Database**: SQLite.
- **AI Symptom Checker**: AI integration to recommend specialists based on symptoms.
- **Authentication**: Secure JWT-based login/registration.

## Installation
1. Clone the repository
2. Run `npm install`
3. Setup an `.env` file with `JWT_SECRET` and `OPENAI_API_KEY` (if required)
4. Start the server with `npm start` or `npm run dev`

## Available Scripts
- `npm start` - Starts the core server
- `npm run dev` - Starts the server using nodemon for development
